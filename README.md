# Fishing

A rainy lotus pond web game built with Three.js. Move the child-sized fishing net with the mouse, touch, or camera motion, scoop goldfish from below, and drop them into the bucket to score.

Public page after GitHub Pages deployment:

https://scsocesong.github.io/fishing/

## Local Development

```powershell
npm.cmd install
npm.cmd run dev -- --port 5173
```

## Build

```powershell
npm.cmd run build
```
